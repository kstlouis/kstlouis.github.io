This is a paragraph. Paragraphs, like headers, are block level elemments; meaning they all require one line. If you placed several headers or parapraphs elements on a single line in your text editor, they would all be rendered on separate lines when viewed via your browser. Not all elements behave this way. This paragraph also has exampled of bolded and italicized text, so you can add more expression to your writing.As it turns out, these are not block level elements; these are called inline elements!

Now, let's do an undordered list!

Just like how you can have bold tags nested inside paragraphs, you can do the same thing in lists. In fact, you can also have lists nested inside other lists!

Divs and Spans

Not super relevant yet, since we don't have any CSS elements to make things look pretty, but here there was a bit of time spent learning about containers. The < div > tag is a block level element, and the < span > is an inline element, but otherwise allow you to do the same thing, for instance change the color/font/size/background/etc of a particular element on the page wihtout having it apply everywhere.
